这是C语言程序设计课程中的课程作业，两个Project是围绕基于遗传算法的自动吃豆人小游戏，其中PJ1实现了手动的吃豆人小游戏，而PJ2则进一步利用遗传算法实现的自动吃豆人
This is a course assignment for the C Language Programming course. The two projects are centered around a Pac-Man mini-game based on genetic algorithms. Project PJ1 implements a manual Pac-Man mini-game, while Project PJ2 further utilizes genetic algorithms to create an automatic Pac-Man.
